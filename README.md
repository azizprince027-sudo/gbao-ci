# GBAO CI 🇨🇮

**GBAO CI** est une plateforme web d'information centralisée pour le système éducatif ivoirien.

## Présentation du Projet

L'objectif de ce site est de réduire la fracture informationnelle en offrant aux élèves et étudiants un accès direct et simplifié aux ressources académiques.

### 🔍 Missions principales :
* **Actualités :** Suivi de l'évolution du calendrier et des réformes scolaires.
* **Opportunités :** Répertoire des bourses d'études nationales et internationales.
* **Orientation :** Informations critiques sur les concours d'entrée aux grandes écoles.

## Public Cible
* **Élèves** du secondaire (préparation aux examens et concours).
* **Étudiants** du supérieur (recherche de bourses et insertion).

## Stack Technique

Pour ce projet, j'ai privilégié une architecture légère et performante :
* **HTML5 :** Utilisation de balises sémantiques (`<header>`, `<main>`, `<section>`, `<footer>`) pour un meilleur référencement (SEO).
* **CSS3 :** Mise en page moderne utilisant **Flex** et **CSS Grid** pour garantir une expérience fluide sur ordinateur et smartphone.

## Aperçu des fonctionnalités
- [x] Interface intuitive et minimaliste.
- [x] Navigation optimisée pour les supports mobiles.
- [x] Organisation de l'information par catégories (Bourses, Concours, News).
